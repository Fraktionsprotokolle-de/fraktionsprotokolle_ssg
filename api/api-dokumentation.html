KGParl Fraktionsprotokolle API

REST API für die Fraktionsprotokolle der Kommission für Geschichte des Parlamentarismus und der politischen Parteien (KGParl)

Version 0.1.0
Kontakt info@kgparl.de
Website www.kgparl.de
Lizenz CC BY 4.0

Server

Produktion https://api.fraktionsprotokolle.de
Entwicklung http://localhost:8080

Info

▶ GET /info API-Informationen abrufen

Gibt grundlegende Informationen über die API zurück.

Antworten

200 Erfolgreiche Antwort
{ "name": "KGParl", "version": "0.1.0" }

Personen

▶ GET /persons Personenliste abrufen

Gibt eine paginierte Liste aller Personen im Register zurück. Unterstützt Suche und Filterung nach Anfangsbuchstaben.

Query-Parameter

Name Typ Beschreibung
page integer Seitennummer (1-basiert). Standard: 1
per_page integer Ergebnisse pro Seite (max. 250). Standard: 50
q string Suchbegriff (durchsucht Nachname, Vorname, Registerbezeichnung). Standard: *
letter string Filter nach Anfangsbuchstaben des Nachnamens
found boolean Filter nach Personen mit/ohne Fundstellen in Protokollen

Antworten

200 Erfolgreiche Antwort
{ "data": [ { "document": { "id": "BrandtWilly_1913-12-18", "surname": "Brandt", "forename": "Willy", "reg": "Brandt, Willy", "letter": "B", "gnd": "118514768", "birth": "1913-12-18", "death": "1992-10-08", "isMDB": true, "found": true }, "highlights": [], "text_match": 0 } ], "pagination": { "page": 1, "per_page": 50, "found": 12847, "total_pages": 257 } }
500 Serverfehler
▶ GET /person/{id} Einzelne Person abrufen

Gibt detaillierte Informationen zu einer Person zurück.

Pfad-Parameter

Name Typ Beschreibung
id * string Eindeutige ID der Person (z.B. BrandtWilly_1913-12-18)

Antworten

200 Erfolgreiche Antwort
{ "id": "BrandtWilly_1913-12-18", "surname": "Brandt", "forename": "Willy", "reg": "Brandt, Willy", "letter": "B", "gnd": "118514768", "birth": "1913-12-18", "death": "1992-10-08", "birth_place": "Lübeck", "death_place": "Unkel", "isMDB": true, "found": true }
404 Person nicht gefunden
500 Serverfehler

Protokolle

▶ GET /calendar Kalendereinträge abrufen

Gibt eine paginierte Liste von Protokollen zurück, optional gefiltert nach Datumsbereich. Geeignet für Kalenderansichten.

Query-Parameter

Name Typ Beschreibung
page integer Seitennummer (1-basiert). Standard: 1
per_page integer Ergebnisse pro Seite (max. 250). Standard: 250
from date Startdatum des Zeitraums (ISO 8601, z.B. 1949-01-01)
to date Enddatum des Zeitraums (ISO 8601, z.B. 1953-12-31)

Antworten

200 Erfolgreiche Antwort
{ "data": [ { "id": "cdu-csu-01_1949-09-07-t0000_EP", "fraction": "CDU/CSU", "startdate": "1949-09-07", "name": "Fraktionssitzung am 7. September 1949", "color": "#000000", "items": [ { "name": "Wahl des Bundeskanzlers", "link": "#div_item_0" } ] } ], "pagination": { "page": 1, "per_page": 250, "found": 4523, "total_pages": 19 } }
500 Serverfehler

Datenmodelle

Person
id string Eindeutige ID der Person
surname string Nachname
forename string Vorname
reg string Registerbezeichnung (Nachname, Vorname)
letter string Anfangsbuchstabe des Nachnamens
gnd string GND-Identifier (Gemeinsame Normdatei)
birth string Geburtsdatum
death string Sterbedatum
birth_place string Geburtsort
death_place string Sterbeort
isMDB boolean War die Person Mitglied des Bundestages?
found boolean Wurde die Person in Protokollen gefunden?
Pagination
page integer Aktuelle Seitennummer
per_page integer Ergebnisse pro Seite
found integer Gesamtanzahl der gefundenen Einträge
total_pages integer Gesamtanzahl der Seiten
CalendarEntry
id string Eindeutige ID des Protokolls
fraction string Name der Fraktion
startdate date Datum der Sitzung
name string Titel des Protokolls
color string Farbcode für Kalenderdarstellung
items array Tagesordnungspunkte
FacetItem
name string Name des Facettenwerts (z.B. Parteiname oder Wahlperiode)
count integer Anzahl der Dokumente mit diesem Wert
Error
error string Fehlermeldung
